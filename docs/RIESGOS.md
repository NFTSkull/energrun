## Riesgos y mitigaciones

- **Dependencia de WhatsApp**: si el usuario no tiene WhatsApp, el flujo puede fallar.
  - Mitigación: mostrar el link completo y/o fallback a copiar mensaje (iteración futura).
- **Validación de teléfono**: formatos variados en MX.
  - Mitigación: normalizar a dígitos y validar longitud mínima razonable.
- **SEO local**: competencia alta (generadores / solar en Monterrey).
  - Mitigación: estructura semántica, metadata, copy con keywords, performance.
